Lightbox component for WPBakery page builder

== Description ==

Painless way to add lightbox component into WPBakery page builder

- Using ultra-lightweight dependencies [lity](https://sorgalla.com/lity/) 
- Minimalize style with BEM Naming, it means more control for customization!
- JS and CSS only loaded when shortcode exist
